# ユビキタス言語定義 (Glossary)

## 概要

このドキュメントは、VRChatイベントカレンダー自動化システムで使用される用語の定義を管理する。
チーム内の認識を統一し、ドキュメント・コード・会話での用語を一貫させることを目的とする。

**更新日**: 2026-03-01

---

## 1. ドメイン用語

プロジェクト固有のビジネス概念・機能に関する用語。

### イベント

**定義**: VRChat 上で開催される催し。本システムの管理対象単位。

**スプレッドシート上の表現**: メインシートの1行が1イベントに対応する。

**関連用語**: イベントID、イベント主催者、イベントジャンル

### イベントID

**定義**: Google Calendar が発行するイベントの一意識別子。スプレッドシートに記録し、更新・削除時の紐付けに使用する。

**コード上の列名定数**: `COL_EVENT_ID = 'イベントID'`

**英語表記**: Event ID

**備考**: カレンダー登録成功時にシステムが自動書き込みする。未登録・削除済みのイベントでは空文字。

### イベント主催者

**定義**: VRChat でイベントを開催し、フォームからイベント情報を登録する人物。

**コード上の列名定数**: `COL_EVENT_ORGANIZER = 'イベント主催者'`

**英語表記**: Event Organizer

**関連用語**: ブラックリスト（主催者名でのフィルタリング対象）

### 修正URL

**定義**: Google Forms が発行する回答編集リンク（editResponseUrl）。登録者が自分のイベント情報を後から変更・削除するために使用する。

**コード上の列名定数**: `COL_EDIT_URL = '修正URL'`

**英語表記**: Edit Response URL / Edit URL

**関連用語**: 修正URL請求

### 修正URL請求

**定義**: イベント主催者が修正URLを紛失した際に、専用フォームからURLの再送を依頼する機能。

**処理関数**: `sendEventEditUrls(e)`

**シート名**: `修正URL請求`（定数: `REQUEST_URL_SHEET_NAME`）

### 変更フラグ

**定義**: カレンダー処理（登録・更新・削除・エラー）が完了した行に立てるフラグ。メール通知の対象判定に使用する。

**コード上の列名定数**: `CHANGE_FLAG_COLUMN_NAME = '変更フラグ'`

**英語表記**: Change Flag

**状態**:
| 値 | 意味 |
|----|------|
| ON（文字列 `"1"` など） | カレンダー処理済み、メール通知待ち |
| 空文字 / クリア | 未処理、またはメール送信完了 |

### 送信済みフラグ

**定義**: メール送信が完了した行に立てるフラグ。二重送信を防止する。

**コード上の列名定数**: `SENT_FLAG_COLUMN_NAME = '送信済みフラグ'`

**英語表記**: Sent Flag

**状態**:
| 値 | 意味 |
|----|------|
| ON | メール送信済み |
| 空文字 / クリア | 未送信 |

### 語録リスト

**定義**: ブラックリスト情報を管理するスプレッドシートのシート。迷惑登録対策として、特定のメールアドレスや主催者名からのイベント登録をブロックする。

**シート名定数**: `BLACKLIST_SHEET_NAME = '語録リスト'`

**英語表記**: Blacklist Sheet

**構成**:
| 列 | 内容 |
|----|------|
| A列 | ブロック対象メールアドレス |
| B列 | ブロック対象主催者名 |

### 迷惑登録

**定義**: ブラックリストに該当するユーザーによるイベント登録。カレンダーへの登録はブロックされ、メール通知も送信されない（サイレントブロック）。

**関連用語**: 語録リスト、ブラックリスト

### Android対応可否

**定義**: イベントが Android 版 VRChat（Meta Quest 等）から参加可能かどうかを示すフォーム項目。カレンダータイトルのプレフィックスに反映される。

**コード上の列名定数**: `COL_ANDROID_PC = 'Android対応可否'`

**値とタイトル加工**:
| フォーム値 | カレンダータイトル |
|-----------|------------------|
| `PC/android` | `【Android 対応】{イベント名}` |
| `android only` | `【Android オンリー】{イベント名}` |
| 上記以外 | `{イベント名}`（加工なし） |

### 削除チェック

**定義**: Google Forms 上で「イベントを削除する」を選択する操作。フォーム再送信時にこのフラグが立っていると、カレンダーからイベントが削除される。

**コード上の列名定数**: `COL_DELETE_CHECKBOX = 'イベントを登録しますか'`

## 2. 技術用語

### Google Apps Script (GAS)

**定義**: Google が提供するサーバーレスのスクリプト実行環境。JavaScript ベースで Google サービス（Forms, Sheets, Calendar, Mail）を操作できる。

**本プロジェクトでの用途**: 全てのバックエンド処理の実行基盤

**ランタイム**: V8

### clasp

**定義**: Google Apps Script のコマンドラインツール（Command Line Apps Script Projects）。ローカルでの GAS コード管理とデプロイを実現する。

**本プロジェクトでの用途**: ローカルのソースコードを GAS プロジェクトに push/pull する

**主要コマンド**:
| コマンド | 用途 |
|---------|------|
| `clasp push` | ローカルコードを GAS にデプロイ |
| `clasp pull` | GAS からローカルにコードを取得 |
| `clasp login` | Google アカウントで認証 |

### Google Calendar

**定義**: Google が提供するカレンダーサービス。本システムのイベント表示先。

**本プロジェクトでの用途**: イベント情報の登録・更新・削除先

**識別子**: `CALENDAR_ID`（環境ごとに異なる）

**公開URL**: https://vrceve.com/

### Google Forms

**定義**: Google が提供するフォームサービス。

**本プロジェクトでの用途**: イベント登録フォーム、修正URL請求フォームの2つを使用

### Google Spreadsheet

**定義**: Google が提供するスプレッドシートサービス。

**本プロジェクトでの用途**: イベントデータのマスターストア。フォーム回答の蓄積、フラグ管理、ブラックリスト管理。

### MailApp

**定義**: GAS でメールを送信するための組み込みサービス。

**本プロジェクトでの用途**: イベント登録結果通知、修正URL送信

**制限**: 1日あたりのメール送信クォータあり（`MailApp.getRemainingDailyQuota()` で残数取得）

### scriptId

**定義**: GAS プロジェクトを一意に識別するID。`.clasp.json` に記録される。

**本プロジェクトでの注意**: 各環境の Form 用・Spreadsheet 用で計6つの scriptId が存在する（既知の問題: dev と prod の Form が同一 scriptId）

## 3. シート名定数

コード内で使用されるシート名の一覧。

| 定数名 | 値 | 説明 |
|--------|-----|------|
| `MAIN_EVENT_SHEET_NAME` | `'フォームの回答 1'` | メインのイベントデータシート |
| `REQUEST_URL_SHEET_NAME` | `'修正URL請求'` | 修正URL請求フォームの回答シート |
| `BLACKLIST_SHEET_NAME` | `'語録リスト'` | ブラックリスト管理シート |

## 4. 列名定数

スプレッドシートのヘッダー行に対応する列名定数。`getColumnMapping()` によって列番号に動的マッピングされる。

| 定数名 | 値（ヘッダー名） | 説明 |
|--------|-----------------|------|
| `COL_EMAIL` | `'メールアドレス'` | 登録者のメールアドレス |
| `COL_EVENT_NAME` | `'イベント名'` | イベントの名称 |
| `COL_EDIT_URL` | `'修正URL'` | フォーム回答の編集リンク |
| `COL_DELETE_CHECKBOX` | `'イベントを登録しますか'` | 削除操作のプルダウン |
| `COL_EVENT_ID` | `'イベントID'` | Google Calendar のイベント識別子 |
| `COL_START_TIME` | `'開始日時'` | イベント開始日時 |
| `COL_END_TIME` | `'終了日時'` | イベント終了日時 |
| `COL_EVENT_ORGANIZER` | `'イベント主催者'` | 主催者名 |
| `COL_ANDROID_PC` | `'Android対応可否'` | Android 対応状況 |
| `COL_EVENT_DETAILS` | `'イベント内容'` | イベントの詳細説明 |
| `COL_EVENT_GENRE` | `'イベントジャンル'` | イベントの分類 |
| `COL_CONDITIONS` | `'参加条件（モデル、人数制限など）'` | 参加条件 |
| `COL_METHOD` | `'参加方法'` | イベントへの参加方法 |
| `COL_REMARKS` | `'備考'` | その他の情報 |
| `CHANGE_FLAG_COLUMN_NAME` | `'変更フラグ'` | カレンダー処理完了フラグ |
| `SENT_FLAG_COLUMN_NAME` | `'送信済みフラグ'` | メール送信完了フラグ |

## 5. 関数名マッピング

コード上の主要関数と役割の対応。

| 関数名 | 所属 | 役割 |
|--------|------|------|
| `onFormSubmit(e)` | Form GAS | フォーム送信時に修正URLをスプレッドシートに記録 |
| `createOrUpdateCalendarEvent(e)` | Spreadsheet GAS | カレンダーイベントの登録・更新・削除 |
| `sendAggregatedEmails()` | Spreadsheet GAS | 変更フラグが立った行の集約メール送信 |
| `sendEventEditUrls(e)` | Spreadsheet GAS | 修正URL請求への応答メール送信 |
| `getColumnMapping(sheet)` | Spreadsheet GAS | ヘッダー行から列名→列番号マッピングを取得 |
| `getBlacklists()` | Spreadsheet GAS | 語録リストからブラックリスト情報を取得 |
| `copyRowToLog()` | Spreadsheet GAS (dev_multi) | 複数登録シートへの行コピー（実験機能） |

## 6. グローバル定数

| 定数名 | 型 | 説明 |
|--------|-----|------|
| `CALENDAR_ID` | String | Google Calendar のカレンダーID（環境ごとに異なる） |
| `NOTIFICATION_CALENDAR_URL` | String | 通知メールに記載するカレンダーURL（`https://vrceve.com/`） |
| `MAX_EMAILS_PER_RUN` | Number | 1回の実行あたりのメール送信上限（デフォルト: 20） |

---

## 7. トリガー種別

| トリガー | 対象サービス | 関数 | 起動タイミング |
|---------|------------|------|--------------|
| フォーム送信時 | Google Form | `onFormSubmit()` | イベント登録フォーム送信 |
| スプレッドシート編集時 | Spreadsheet | `createOrUpdateCalendarEvent()` | フォーム回答がスプレッドシートに追記された時 |
| 時間ベース（定期） | Spreadsheet | `sendAggregatedEmails()` | 設定された時間間隔で定期実行 |
| フォーム送信時（請求） | Spreadsheet | `sendEventEditUrls()` | 修正URL請求フォーム送信 |

---

## 8. メール通知パターン

| パターン | 件名 | 送信条件 |
|---------|------|---------|
| 登録・更新完了 | イベント登録・更新完了通知 | バリデーション通過、カレンダー登録/更新成功 |
| 削除受付 | イベント削除申請受付 | 削除チェック選択による削除完了 |
| 過去日時エラー | 登録失敗通知 | 開始日時が過去 |
| 日時逆転エラー | 登録失敗通知 | 終了日時 < 開始日時 |
| 修正URL送付 | （修正URL一覧） | 修正URL請求への応答 |
| 該当なし | （該当なし通知） | 修正URL請求で対象イベントなし |
| 迷惑登録 | （送信なし） | ブラックリスト一致時はサイレントブロック |

| 環境名 | ディレクトリ | 用途 |
